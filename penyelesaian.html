Soal ke-1: Tentukan nilai dari 3log27.

Penyelesaian:

3log27 adalah pangkat yang harus diberikan pada 3 untuk mendapatkan 27. Karena 33 = 27, maka 3log27 = 3.

Soal ke-2: Sederhanakan ekspresi alog(x2y) - alog(x).

Penyelesaian:

  • alog(xm) = malogx
  • alog(x/y) =a logx - alogy

Maka:

alog(x2y) - alog(x) = alog(x2) + alog(y) - alog(x) = 2alogx + alogy - alogx = alogx + alogy = alog(xy)

Soal ke-3: Hitung nilai 2log5 jika diketahui 10log2 = 0.3010 dan 10log5 = 0.6990.

Penyelesaian:

Menggunakan rumus perubahan basis logaritma:

alogb = clogb / cloga

Maka:

2log5 = (10log5)/10log2 = 0.6990 / 0.3010 ≈ 2.322

Soal ke-4: Tentukan nilai x yang memenuhi persamaan 2log(x + 1) = 3.

Penyelesaian:

Menggunakan definisi logaritma, kita ubah persamaan ke bentuk eksponensial:

23 = x + 1

Maka:

8 = x + 1

x = 7
